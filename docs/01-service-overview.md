# 1. サービス概要

## サービス名（仮）

**ThermoInspect Report** — ドローン赤外線点検 報告書自動作成 Web アプリ

## 背景・課題

ドローンを用いた赤外線（サーマル）点検は、太陽光パネル・外壁・屋根などの異常検知に広く使われている。しかし現場では以下の課題がある。

- 撮影画像の整理・管理が煩雑
- 異常箇所の記録が Excel や紙ベースで属人化している
- 報告書作成に数時間〜数日かかる
- 異常所見のコメント作成に専門知識と時間が必要
- 報告書フォーマットが点検会社ごとにバラバラ

## サービスコンセプト

**「点検データから、工事受注率を上げる営業支援型報告書を自動生成する」**

単なる調査報告書ではなく、**総合評価スコア・残寿命推定・推奨工事プラン** を含む 3 種類の PDF（調査版 / 顧客提出版 / 営業提案版）をワンクリックで出力。AI が異常ごとの診断コメントを自動生成し、営業提案まで一気通貫で支援する。

点検オペレーターが Google アカウントでログインし、点検種別を選んでプロジェクトを作成。赤外線画像をアップロード、異常箇所を記録すると、システムが **A〜E 評価・緊急度・推奨対応時期** を自動算出。営業提案版 PDF で工事プランまで提示できる。

## ターゲットユーザー

| ユーザー | 説明 | role |
|----------|------|------|
| **点検オペレーター** | ドローンで赤外線撮影を行い、報告書を作成する技術者 | `operator` |
| **管理者（Admin）** | チーム・ユーザー管理、全ユーザー閲覧 | `admin` |

> MVP では点検オペレーター + 管理者の 2 ロールに限定する。  
> 発注者（クライアント）向け閲覧機能は Post-MVP。

## 対応点検種別（MVP）

| コード | 名称 | 主な異常 |
|--------|------|----------|
| `SOLAR_PANEL` | 太陽光パネル | ホットスポット、層間剥離、クラック |
| `EXTERIOR_WALL` | 外壁 | 断熱不良、水分侵入、クラック |
| `ROOF` | 屋根 | 雨漏り兆候、断熱不良、劣化 |

## 提供価値

1. **工事受注率の向上** — 営業提案版 PDF で推奨工事プランまで自動提示
2. **自動評価** — 総合スコア A〜E、緊急度★、推奨対応時期を自動算出
3. **残寿命・リスク可視化** — 屋根残寿命推定、太陽光発電リスク表示
4. **AI 診断コメント** — 異常ごとに所見・リスク・推奨アクションを自動生成
5. **3 種 PDF** — 調査版 / 顧客提出版 / 営業提案版を用途別に出力
6. **作業時間の短縮** — 報告書作成を数時間 → 数分に
7. **データの一元管理** — 画像・異常・評価・報告書をクラウド上で管理

## 技術スタック

| レイヤー | 技術 |
|----------|------|
| フロントエンド | React + TypeScript + Vite + TailwindCSS |
| バックエンド | Node.js + Express + TypeScript |
| データベース | PostgreSQL |
| 認証 | Google OAuth 2.0 + JWT（HttpOnly Cookie） |
| AI | LLM API（診断コメント生成） |
| ファイルストレージ | ローカル（MVP）→ S3 等（Post-MVP） |
| PDF 生成 | Puppeteer / pdfkit（Phase 4 で選定） |

## 非機能要件（MVP）

| 項目 | 目標 |
|------|------|
| 同時ユーザー | 〜10 名 |
| 画像アップロード | 1 回あたり最大 50 枚 / 各 20 MB（JPEG/PNG/HEIC） |
| 対応ブラウザ | Safari（macOS/iOS）/ Chrome / Edge 最新版 |
| AI コメント生成 | 10 秒以内 |
| 報告書生成 | 30 秒以内 |
| 可用性 | 開発環境のみ（本番は Post-MVP） |

## MVP 対象外

- **Stripe 決済** — 課金機能は Post-MVP（MVP では全機能無料）
- AI 自動異常検知（画像解析）
- 組織（テナント）管理
- 本番デプロイ
